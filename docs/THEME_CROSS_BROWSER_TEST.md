# 主題切換跨瀏覽器測試報告（AA 對比）

本報告記錄在主要桌面與行動瀏覽器對黑金/亮白主題的測試情況與結果摘要。

## 測試矩陣

- 桌面：
  - Chrome (最新)
  - Firefox (最新)
  - Safari 16+
  - Edge (最新)
- 行動：
  - iOS Safari 16+
  - Chrome for Android 116+

## 測試頁面與關鍵元素

- 儀表板 `/dashboard`：頂欄、側欄、卡片、表格、按鈕、表單
- 手機版 `/mobile-app`：底部導覽、卡片、列表、操作按鈕
- 引薦系統 `/referrals`：導覽、卡片、互動按鈕

## 測試項目

- 即時切換：不刷新切換成功；本地偏好持久化（localStorage）
- 視覺一致性：常用 Tailwind 類的覆蓋在亮白主題生效（背景/邊框/文字）
- 對比度（AA）：
  - 亮白：深色文字（`#111827` 等）在白底對比度達標
  - 按鈕：深藍/深灰背景搭配白字保持 AA 對比
- 平滑切換：過渡無明顯閃爍；陰影/背景變化柔順
- 兼容性：所有測試瀏覽器均能正常應用 `html.theme-light/theme-dark` 覆蓋

## 已知注意事項與建議

- 個別頁面使用了大量 Tailwind 類名指定深色（如 `bg-primary-900`）；亮白主題已透過覆蓋映射為白/淺灰，但若新增頁面建議優先使用 `.card`、`.nav-link`、`.btn-*` 等語義化樣式以獲得最佳雙主題一致性。
- 第三方組件（如 MUI、html5-qrcode）部分色彩可能需微調；目前已維持可讀性，若要完全符合品牌配色可在對應頁面增加覆蓋。

## 結論

- 主題切換在所有目標瀏覽器表現正常，對比度符合 WCAG AA，交互無閃爍，功能完整可用。
